Lista de iconos

Puede obtener una lista de los iconos completos en Bootstrap Icons.

Para agregar a su aplicación, todos los nombres de los íconos se transformaron en camel case usando lodash, siempre comenzando con la primera letra en mayúsculas.

Por ejemplo, para usar el icono bell-fill:

import { BellFill } from 'fleet-icons/icon/BellFill';
const App = () => {
return (
<BellFill />
)
}
