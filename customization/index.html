Customization

With styled-components, you can comfortably customize the styles of the icons.

Example#

import styled from 'styled-components';
import { TrashFill } from 'fleet-icons/icon/TrashFill';
const TrashFillIcon = styled(TrashFill)`
fill: #673AB7;
width: 2em;
height: 2em;
transition: all .2s;
&:hover {
fill: #FF5722;
width: 2.5em;
height: 2.5em;
}
`;
export TrashFillIcon;
